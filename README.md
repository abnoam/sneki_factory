# 🏭 Sneki

### Food Factory Inventory & Order Management System

A Java-based system for managing raw materials, products, warehouse inventory, and customer orders.

## ✨ Features

- 📦 Inventory Management
- 🥫 Raw Material Tracking
- 🛒 Customer Order Processing
- 🚚 Distributor Management
- 📅 Expiration Date Monitoring
- 💰 Profit & Value Calculations
- 🌳 Binary Search Tree Inventory Storage
- 🔄 Queue-Based Order Management

---

## 🏗️ Architecture

```text
Client
 ├── Distributor

RawMaterial (Abstract)
 ├── LiquidRawMaterial
 └── SolidRawMaterial

Product
 └── RawMaterial List

Order
 └── Product List
```

## 🛠️ Built With

- Java
- Object-Oriented Programming
- Java Collections Framework
- Binary Search Trees (BST)
- Queues
- Linked Lists
